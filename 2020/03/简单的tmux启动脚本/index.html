简单的tmux启动脚本

三月 27, 2020 发布在 shell脚本

可用于脚本tmux命令

创建一个命名为 workspace 的会话，并且命名首窗口为 editor > tmux new-session -s workspace -n editor 关联到命名窗口 > tmux attach -t workspace 向命名会话 workspace 中的活动窗口或窗格发送按键，C-m 相当于回车键 > tmux send-keys -t workspace ‘[keys]’ C-m 向命名会话 workspace 中的第 1 个窗口中的第 0 个 窗格发送按键，C-m 相当于回车键 > tmux send-keys -t workspace:1.0 ‘[keys]’ C-m 使命名会话 workspace 中的第 1 个窗口成为活动窗口 > tmux select-window -t workspace:1 使命名会话 workspace 中的第 1 个窗口成为活动窗口 将命名会话 workspace 中的当前窗口垂直分割，其中下面的窗格高度占 10% > tmux split-window -v -p 10 -t workspace
为命名会话 workspace 选择布局 > tmux select-layout -t workspace main-horizontal
加载 app.conf 配置文件，并关联到该配置文件生成的会话 > tmux -f app.conf attach

简单的启动

  • 创建2个workspace，打开nvim ```shell session=“workspace”

    $session

    tmux new-session -s workspace -n editor -d

    tmux send-keys -t $session ‘cd /Users/gmtcover/Learn’ C-m

    tmux send-keys -t workspace ‘cd ~/Learn/’ C-m tmux send-keys -t workspace ‘nvim’ C-m tmux split-window -v -p 30 -t workspace

    tmux send-keys -t workspace:1.2 ‘cd ~’ C-m

tmux send-keys -t workspace:1.2 ‘date’ C-m tmux new-window -n console -t workspace

tmux send-keys -t workspace:2 ‘cd ~’ C-m

tmux select-window -t workspace:1

tmux new-window -t $session:2 -n edit nvim

tmux select-window -t workspace:1.1 tmux attach-session -t workspace
