内存数据存放位置

  • 本地变量在栈stack
    • 编译器自动分配释放
  • new malloc 放在堆里面 heap
    • 程序员分配释放
  • 全局区（静态区）stctic
    • 初始化的全局变量和静态变量在一块区域，为初始化的在相邻的一块区域，程序结束自动释放
  • 文字常量区
    • 常量字符串，程序结束系统自动释放
  • 程序代码区
    • 存放函数体的二进制代码

  • 内存是虚拟的独立的，有内存管理单元，内存保护机制
