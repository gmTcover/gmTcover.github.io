C语言整理

九月 15, 2019 发布在 C基础, pointer

c指针

  • 指针的类型是deriv from 其他类型，也就是说指针的类型是由他指向的类型决定的
  • 指针是一种reference类型，即引用类型

  • 指针就是地址

    int main()
{
	int foo;
	int *foo_p;

	foo = 5;
	foo_p = &foo;

	printf("   foo...%d\n",foo);
	printf(" foo_p...%d\n",*foo_p);
	printf("  &foo...%p\n",&foo);
	printf(" foo_p...%p\n",foo_p);
	printf("&foo_p...%p\n",&foo_p);
	return 0;
}

几点说明

  • %p 是专门用于指针变量中的内容，虽然与%x结果完全一样但是含义完全不同。%p用合适的方式（一般16进制）输出指针变量中存放的另一个变量的的地址，%x用16进制的方式打印出变量的值，又是会省去地址前面的0. > 插入图片 指针变量的示意图
  • 指针是一种特殊的语言机制，类似本地变量，有自己的地址，该地址上内容为其他变量的地址，通过运算符获取改地址内容。

高阶指针

  • 指向指针的指针

    int main()
{
	int foo =5;
	int *foo_p = &foo;//定义一个指针，类型为int foo_p=&foo
	int **foo_pp = &foo_p;//定义一个指向指针的指针，指针间建立关系
	//foo_pp = &foo_p,指针上的内容为地址

	printf("    foo...%d\n",foo);

	printf("   &foo...%p\n",&foo);
	printf("  foo_p...%p\n",foo_p);

	printf(" &foo_p...%p\n",&foo_p);
	printf(" *foo_p...%d\n",*foo_p);

	printf(" foo_pp...%p\n",foo_pp);//%p打印指针上面的值


	printf("&foo_pp...%p\n",&foo_pp);
	prntf("*foo_pp...%p\n",*foo_pp);//得到的是指向指针的地址上的值（是一个地址）
	printf("**foo_pp..%d\n",**foo_pp);
	return 0;
}

  • 定义指向指针的指针模型，两个指针建立关系，高阶指针上面的内容为前一个指针的地址

  • 指针是派生类型，它的类型依靠它所指向的对象

c语言中的语法声明

  • 原则1:先看表示符
  • 原则2:运算符优先级
  • 指针数组：首先他是一个数组，数组元素都是指针，数组占多少个字节由数组本身决定，它是“储存指针的数组”的简称
  • 数组指针：首先他是一个指针，它指向一个数组。至于它指向的数组占多少字节，不知道。它是“指向数组的指针”的简称
