字符与字符串

九月 23, 2019 发布在 category, subcategory

字符数组

char word[] = {'H','e','l','l','o','!'};

  • 这个不是c语言的字符串，不能用字符串的方式做计算

    char word[] = {'H','e','l','l','o','!','\0'};

  • 有7个单元（‘\0’），使word成为c语言字符串，可以使用字符串运算

  • 结尾可以使0（4byte），\0（一定为一个字节）

    • 0标志字符串的结束，当它不是字符串的一部分

  • string.h头文件函数，基于有\0构造

    字符串变量

    char *str = "Hello";//栈区
char word[] = "Hello";
char line[10] = "Hello";

    字符串常量（字面量）

  • “。。。”双引号之内

    • 会被编译器变成一个字符数组存放在某处，包括结尾\0，（栈区）

      printf("Ah, the user with no name. "
	"Well, ok, hi, user with no \"name\":");

  • c语言自动把字符串连接

  • c语言字符串是以字符数字的形态存在

    • 不能用运算符对字符串做运算
    • 通过数组的方式可以遍历字符串

  • 唯一特殊的地方是字符串字面量可以用来初始化数组（自动编译成字符串数组）

    字符串常量

    char* s = "Hello, world!";

//实际上类型为const char* s,read-only，无法修改

  • 需要可以修改的字符串

    char s[] = "Hello, world!";

  • 区别，数组定义字符串就在我这里(实际上通过外部read-only复制过来到本地变量区)，因为在本地，可以修改

    • 指针定义指向的是read-only的。

      如何选择？指针还是数组

  • 数组：作为本地变量可以修改，空间自动回收–》构造一个字符串

  • 指针：参数处理（仅只读），动态分配空间–〉处理一个字符串

    字符串赋值

    char* t = "title";
char* s ;
s = t;

  • 定义来一个新指针指向字符串而已，没有产生新的字符串。

    字符串的输入输出

    char string[8];
char str[8];
scanf("%7s",string);
scanf("%s",str);

printf("%s\n",string);

printf("%s%s",string,str)
//安全输入
//c语言处理：scanf_s("%s",str,8);
// scanf("%7s",word);
//c++处理，cout >> setw(nm_size) >> word; 头文件iomanip

  • 只会读取单个单词，读到空格，回车为止，之后内容不会输出

  • 不安全越界

    常见错误

  • 用指针定义字符串

    char *string;

scanf("%s",string);

//没有初始化，指针上原有内容行被写入修改，导致运行出错。

  • 空字符串

    char buff[100] = "";//这是一个空字符串，只有buff[0] = '\0'

char buff[]="";//这个字符串长度只有1

字符串数组

char **a;//a是一个指针，指向一个char* 类型的指针（指向定义好的字符，也可以是字符串，首地址）

char a[][10];//二维数组但是第二个括号没有完备的类型

char *a[];//数组，其中每个元素为指针类型为char* +首地址，指向内容外部某处存储read-only内容（字符，字符串）
int main()
{
char a[][10] = {
	"Hello",
	"World!",
}

int main()
{
char *a[] = {
	"Hello",
	"World!",

}
}
- 指针是malloc动态分配一段空间，只读,写入时需要初始化。传的首地址+类型
- 数组本地变量从，read-only复制过来需要定义大小，不能越界。
```#include <stdio.h>
int main()
{
	int num;
	char *a[]= {"Jan","Feb","Mac","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"
	};
	
	char **b = NULL;
	b =a ;
	printf("Please enter the month：\n");
	
	scanf("%d",&num);
	
	printf("%s\n",b[num-1]);

	return 0;

//**b，不能直接写入，二维数组定义一定要有明确类型char *[]
// **b,可以指定定义好的数组中的任意行。
