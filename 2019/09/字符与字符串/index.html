字符与字符串

九月 23, 2019 发布在 C基础

字符数组

char word[] = {'H','e','l','l','o','!'};

  • 这个不是c语言的字符串，不能用字符串的方式做计算

    char word[] = {'H','e','l','l','o','!','\0'};

  • 有7个单元（‘\0’），使word成为c语言字符串，可以使用字符串运算

  • 结尾可以使 数字0（4byte），字符串‘\0’（一定为一个字节）

    • 0标志字符串的结束，当它不是字符串的一部分

  • string.h头文件函数，基于有\0构造

    字符串变量

    char *str = "Hello";
char word[] = "Hello";
char line[10] = "Hello";

    字符串常量（字面量）

  • “。。。”双引号之内

    • 会被编译器变成一个字符数组存放在某处，包括结尾\0，（栈区）

      printf("Ah, the user with no name. "
	"Well, ok, hi, user with no \"name\":");

  • c语言自动把字符串连接

  • c语言字符串是以字符数组的形态存在

    • 不能用运算符对字符串做运算
    • 通过数组的方式可以遍历字符串

  • 唯一特殊的地方是字符串字面量可以用来初始化数组（自动编译成字符串数组）

    字符串常量

    char* s = "Hello, world!";

//实际上类型为const char* s,read-only，无法修改

  • 需要可以修改的字符串

    char s[] = "Hello, world!";

  • 区别，数组定义字符串就在我这里(实际上通过外部read-only复制过来到本地变量区)，因为在本地，可以修改

    • 指针定义指向的是read-only的。

      字符串的含义

      #include <stdio.h>
int main()
{
	char* s = "Hello";
	char a[][10]={
		"World!",
		"again!"
	};

	printf("%p\n","World");//外部只读
	printf("%p\n",a[0]);//自变量,实际上从只读赋值到本地
	printf("%p\n",a[1]);//地址相差10
	printf("%s\n",a[0]+1);//输出为orld！

	printf("%p\n",s);//输出指针上的内容，相同
	printf("%p\n","Hello");//相同
	
	printf("%d\n",*s);
	printf("%d\n",*"Hello");

	printf("%p\n",&"Hello");//相同
	printf("%p\n",&s);

	printf("%s\n", s);
	printf("%s\n", s+1);//ello
	printf("%s\n","Hello");
	return 0;
}
//指针模型
//字符串类型就是char*，类似一个指针模型（内容和自身&地址相同）,指针内容相同，%s解析出来字符串也相同
//指针s类型为cha*，指向类型为char

如何选择？指针还是数组

  • 数组：作为本地变量可以修改，空间自动回收–>构造一个字符串

  • 指针：参数处理（仅只读），动态分配空间–>处理一个字符串

    字符串赋值

    char* t = "title";//令内容相等
char* s ;
s = t;//都指向相同内容

  • 定义来一个新指针指向字符串而已，没有产生新的字符串。

    字符串的输入输出

    char string[8];
char str[8];
scanf("%7s",string);
scanf("%7s",str);

printf("%s\n",string);

printf("%s%s",string,str)
//安全输入
//c语言ide处理：scanf_s("%s",str,8);
// scanf("%7s",word);
//c++处理，cout >> setw(nm_size) >> word; 头文件iomanip

  • 只会读取单个单词，读到空格，回车为止（shell），之后内容不会输出

  • 不安全越界

    常见错误

  • 用指针定义字符串

    char *string;

scanf("%s",string);

//没有初始化，指针上原有内容行被写入修改，输出不相关内容，导致运行出错。

  • 空字符串

    char buff[100] = "";//这是一个空字符串，只有buff[0] = '\0'

char buff[]="";//这个字符串长度只有1

字符串数组

char **a;//a是一个指针，指向一个char* 类型的指针（指向定义好的字符，也可以是字符串，首地址）

char a[][10];//二维数组但是第二个括号需要完备的类型

char *a[];//模型，数组，其中每个元素为指针类型为char* + 传首地址，对应指向内容外部某处存储read-only内容（字符，字符串）,
int main()
{
char a[][10] = {
	"Hello",
	"World!",
}

int main()
{
char *a[] = {
	"Hello",
	"World!",

}
}
- 指针是malloc动态分配一段空间，只读,写入时需要初始化。（c字符串常量有初始作用）
- 数组本地变量从，read-only复制过来需要定义大小，不能越界。
​```#include <stdio.h>
int main()
{
	int num;
	char *a[]= {"Jan","Feb","Mac","Apr","May","Jun","Jul","Aug","Sep","Oct","Nov","Dec"
	};
	
	char **b = NULL;

// **b,可以指定定义好的数组中的任意行。

程序参数

总结

  • 字符串写出来就在全局空间，只读，类型char*，类似一个指针。用指针或者数组创建（类型相同），赋值以后，传递的是首地址，指针上字符串代表的首地址，%s都能解析出字符串。（大概的过程，还有很多细节）
  • 多维字符串，定义的类型要一定有规模（二级指针不行，区别,w），弄清首地址，类型，指向类型
  • 定义新指针的意义，可以任意定位想表示的位置，使之有弹性。
