结构类型所占的内存与地址

struct test
{
	char a;
	int b;
};
int main()
{
	struct test Test = {'a',1};

	printf("char a地址%p\n",&Test.a);
	printf("int b的值%p\n",&Test.b);

	printf("test的地址%p\n",&Test);

	printf("sizeof(Test):%lu\n",sizeof(Test));

	return 0;

}
  • a，b地址相差3字节，Test 8个字节，存在3个字节的内存孔隙
    • 复杂结构嵌套是c是怎么设计内存
    • 结构体内存对齐
  • 在内存访问寻址不能从任意出开始，所以我能在图里定义一些确定的访问位置

  • 数据紧靠着存放，cpu要读取下一个数据，需要把之前的数据读取一次，内存中充满巨量变量数据时候，cpuu就需要频繁访问内存，读取数据性能大大下降，耗费时间。

    • 空间换取时间

    • 让占用空间小的成员尽量集中在一起

联合（union）

  • 存储
    • 所有成员共享一个空间
    • 同一时间只有一个成员是有效的
    • union的大小是最大的元素

  • 初始化

    • 对第一个成员做初始化

      uion使用输出进制

      typedef union {
	int i;
	char ch[sizeof(int)];//int 的字节大小不同平台不同，char 1字节
}CHI;

int main(int argc, char const *argv[])
{
	CHI chi;
	int i;
	chi.i = 1234;
	for ( i=0; i<sizeof(int); i++){
		printf("%02hhX",chi.ch[i]);
	}
	printf("\n");

	return 0;
}
// 一个int是 4 字节,2个16进制
// 00 00 04 D2，但是计算机cpu机制是x86规格小端机器，内存排列方式：低位在前D2 04 00 00
// 所以真实底部表示方式是 D2 04 00 00
// 通过union 得到一个整数内部的各个 字节，double float，一个转换工具，用途，当做文件操作的时候，当要把一个整数以二进制的形式写到一个文件里时，可以做一个中间的媒介
