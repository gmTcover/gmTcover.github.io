Vim指令

九月 15, 2019 发布在 tool,

vim常用指令

normal 模式

  • gi #快速回到最后编辑位置
  • h j k l

  • w/W b/B #单词间移动

    行间搜索选中想要发字符

    水平移动

  • 0/^ $/g #移动到行首/行首字母；行尾/行尾前一个单词

  • （） #括号句子间移动

  • {} #段落间移动

    vim页面移动

  • gg/G #移动到文件开头结尾，ctrl+o 返回初始位置

  • H/L/M #head,meiddle,lower

  • zz #屏幕居中

  • ctrl+u/ctrl+f #upward/forward翻页

    进入insert模式

  • a button #append

  • o button #open a line blew

  • i button #insert

  • A

  • I

  • O

    替换搜索

    撤销

  • u/U ctrl+r

    精确替换

  • ：% s/ <>/<>

    多文件操作buffer

  • Buffer window Tab ##内存缓存区，窗口，整个工作区域

    多个buffer

  • ：ls #列举当前缓存区

  • ：e 文件名，#打开多个文件，就会有多个缓存区

  • ：b name或p，n，l #文件，缓存区跳转，previous，next，last，Tab补全名字

    多个window

  • ：sp 水平

  • ：vp 垂直

  • ：不同窗口打开多个buffer，一个buffer打开多个窗口或多个buffer多个窗口一个界面

  • ：ctrl+W+H J K L #大写调换窗口位置

  • ：ctrl+h j k l。#小写光标位置在窗口切换

    Tab（标签页）操作

  • ：可以整理文件类型编辑 txt，mid,etc 分类编辑

  • :tabnew #新的标签页

  • gt #标签页切换 类似：b buffer-name

    vim 的text object（文本对象）

  • [number]command[text object] #格式command 是命令，d（delect），c（change），y（yank）number表示次数，object表示对象，比如单词w，句子s，段落p

    [text object]的内容修改

  • i（）；a（）；i<>;a<> ;#还是类似的[];{};“;’ vi+加符号

  • iw # inner word,比如viw 进入visual模式iw选中当前单词

  • aw # a word,选中单词切包含单词后空格

  • is #选中句子

  • as

  • ip #选中段落

  • ap

    选中内容增删改查进入visual模式v选中内容

  • c*[ ;c{;c”;c’ #change内容进入insert

  • x #删除字符

  • d #删除文本对象,并留在寄存器里

  • r

  • s

    Normal 模式复制粘贴

  • y(yank);（put）

  • 配合文本对象：yiw 复制一个单词，yy复制一行,到无名寄存器

  • v命令选中所需位置

    inser 模式复制粘贴

  • ctrl+c ctrl+v

    寄存器中复制粘贴

  • 复制，剪切到不同 寄存器中，可以批量选择地复制，剪切

  • “a yy ；”b yy #复制到不同寄存器a，b

  • ：reg #查看寄存器内容

  • ”a p；“b p 粘贴不同寄存器内容

  • ” yy，#默认寄存器 相当于yy

  • ###专用寄存器

  • 复制专用寄存器“0 使用y复制文本到默认寄存器同时会被拷贝到寄存专用寄存器”0

  • 系统剪切板“+ # 复制出vim中内容到别处

  • ctrl+r{register}多次粘贴一个寄存器的内容

  • 寄存字母有a-z，还有特殊的

    vim宏操作(macro)

  • 重复相同操作，一系列命令的操作，录制回放

  • normal模式中使用q来录制，同时也是q录制结束

  • 使用q{register}选择要保存的寄存器，保存录制的命令

  • 使用@{register}回放寄存器中保存的命令

  • 针对实际问题设计

  • qa（开始）I“ A” Esc q（结束） @a 回放寄存器内容

  • shift+v=V #行选配合 gg/G 文章开头结尾;h j k l移动, 批量选择然后按：进入命令模式进行操作

    批量操作（类似宏）

  • v （选择范围） ：（进入命令行模式） normal（执行normal命令）I“

  • v （选择范围） ：（进入命令行模式） normal（执行normal命令）A”

    插入文件名，文件路径

  • :r! echo % # ctrl+xf 打开文件名

  • :r! ecgo %:p self. #搜索文件中关键字补全 ctrl+n ctrl+f ，n，p（next，previous）

  • 未装插件都是基于文本的匹配补全

    文件目录树

  • 插件 nerdtree

  • ctrl+v/g ; q #打开目录，关闭目录

  • ctrl+v,ctrl+wp #副窗口和目录间切换

    窗口最大化与窗口等宽

  • ctrl+W+| ;ctrl+w+=

    快速定位插件

  • 插件 ctrlp

  • ctrl+p

    快速移动

  • 插件 easymotion

  • ss 命令

    替换符号

  • normal模式选中 r替换+内容

    快速修改符号

  • 插件 vim-surround

  • ds (delete a surrounding)

  • cs (change a surrounding)

  • ys (you add a surrounding )

  • “name” ys + iw +符号#ys 自动进入选中模式了

  • “name” # cs “’ 改变双引号为单引号

  • （）改[] #cs （ 】左边圆括号右边方括号不会产生空格符号

  • ds +” ‘ ( { [

    模糊搜索批量替换

  • 插件fzf.vim

    变成语言插件

    -vim.go插件 #基本可以满足日常开发

  • 需要的功能代码补全，重构，跳转，自动格式化，自动导入等功能。

  • 插件

  • pathon-mode具备基本的补全，跳转，重构格式化功能。

  • 即使是在服务器上也能愉快的调试payhon代码

    tagbar

  • 插件

  • 大项目浏览代码结构，并且搜索跳转到对应代码块

    高亮感兴趣的单词，浏览代码

  • 浏览代码的时候经常需要知道一个变量的使用方式。

  • vim-interestingword

    异步补全插件

  • 支持多变丑语言，支持模糊匹配

  • deoplete

    coc.vim

  • 支持异步补全切支持LSP

  • [](https://github.com/neoclide/coc.nvim)国内大神开发

    代码格式话与静态检查

  • 把精力集中代码逻辑本身，而不是整理格式这些浪费精力的事情上，静态检查时为了让编写的代码更加规范，几十发现bug，不规范

  • 无论你编写何种变丑语言，最好都加上自动格式化和静态检查，更加容易维护。

  • vim-autoformat

  • 需要安装对应语言的格式化库。python autopep8/js的prettier等

    静态检查Lint

  • neomake和ale是两种常用的lint插件

  • [](https://github.com/w0rp/ale) #ale，是代码格式更加干净，有些团队上线前会检查

    代码快速注释

  • vim-commentary

  • 常用命令 gc 注释和取消注释，插件会根据不同的文件类型使用不同注释

  • gc gcc注释掉一行

    git的包装器Fugitive

  • 在vim的git项目 中使用git命令

  • [](https://github.com/airblade/vim-gitgutter)

  • gitgutter

  • [gv.vim]
