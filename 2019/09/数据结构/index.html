数据结构

九月 27, 2019 发布在 C基础, struct

struct

  • 通常函数外部声明类型
    • 块内声明，考虑作用域，生存期

  • 声明结构类型，定义结构变量

    • 变量初始化

      struct date {
		int month;
		int day;
		int year;
};
int main(int argc, char *argv[] )
{
	struct date today = {9, 26, 2019};
	struct date ThisMonth = {.month = 7, .year = 2019 };
	
	printf("Today's date is %i-%i-%i.\n",today.year, today.month, today.day ); 
	printf("Today's date is %i-%i-%i.\n",ThisMonth.year, ThisMonth.month, ThisMonth.day ); 

	return 0;
}

结构运算

  • 要访问整个结构，直接用结构变量的名字

  • 对于整个结构，可以做赋值，取地址，也可以传递给函数参数

    pl =(struct date){9, 26 , 2019 };//强制类型转换
pl = pm;// pl.day = pm.day;

    • 数组无法做运算，const

      结构指针

  • 和数组不同，结构变量的名字并不是结构变量的地址，必须使用&运算符

    struct date *pDate = &today;//指向today变量的指针

    结构与函数

  • 结构作为函数参数，整个结构可以作为参数的值传奇函数（封装了）

    int numberOfdays( struct date today );

  • 函数内新建一个参数类型的结构变量，并复制调用传入参数结构的中的值

  • 也可以返回一个结构

    输入结构

  • 没有直接的方式可以一次scanf 一个结构

  • 不是基础类型，已经代码写好，需要自己写

    struct point {
	int x;
	int y;
};

struct point getStruct(void);// 获得一个结构的函数,结构的传递，废空间也费时间
void output( struct point );

int main(int argc, char const *argv[])
{
	struct point y = {0,0};
	y = getStruct();//结构体赋值运算
	output(y);

	printf("%d, %d\n",y.x, y.y);//修改不了y的值
	return 0;
}

struct point getStruct(void)//传入的是mian函数结构体的复制体，定义好返回类型，
{
	struct point p;//重复定义
	scanf("%d",&p.x);
	scanf("%d",&p.y);
	printf("%d, %d\n",p.x, p.y);
	return p;
}

void output(struct point p)
{
	printf("%d, %d\n",p.x, p.y);
}
//c语言中结构通传递指针，首地址
//struct date *p = &myday;
//(*p).month = 12;
//p->month = 12;//两种方式表示，p所指的那个结构的month 新的运算符，左边是指针

优化结构的输入输出

  • 指针方式传结构

    struct point {
	int x;
	int y;
}y;
struct point* getStruct(struct point*);
void output(struct point);
void print(const struct point *p);

int main(int argc, char const *argv[])
{
	//struct point y = {0,0};
	getStruct(&y);//得到结构指针，
	output(y);
	output(*getStruct(&y));//&y计算后的&y被连续调用,并打印出来值，*运算符获取变量名
	print(getStruct(&y));
	*getStruct(&y) = (struct point){1,2};//连续调用，被重新赋值
	output(y);
}
struct point* getStruct(struct point *p)
{
	scanf("%d",&p->x);
	scanf("%d",&p->y);
	printf("%d, %d\n",p->x, p->y);
	return p;
}
void output(struct point p)// 结构
{
	printf("%d, %d\n",p.x, p.y);
}

void print(const struct point *p)
{
	printf("%d, %d\n",p->x,p->y);

}

总结

  • c语言中结构的传递是值的传递
